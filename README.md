Library Management System (MERN Stack)
A simple web application to manage library books, student requests, and borrowing/returning system.

Features
🧑‍🎓 Student registration & login
📚 Add, update, and delete books
🔍 Search books by name or category
📝 Borrow and return book requests
🧾 Admin approval system
📊 View student profiles and borrowed books

Tech Stack
Frontend: React.js, React Router, Axios
Backend: Node.js, Express.js
Database: MongoDB
Others: Mongoose, Toastify, CORS, dotenv
